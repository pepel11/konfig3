# Ассемблер УВМ (Вариант 20)

## Синтаксис
- Одна команда на строку
- Комментарии через `;`
- Регистр не важен
- Формат: `МНЕМОНИКА аргументы`

## Команды

| Мнемоника | Назначение | A-код | Размер | Аргументы |
|-----------|------------|-------|---------|-----------|
| `LOAD_CONST` | Загрузка константы | 12 | 4 байта | `значение` |
| `READ_MEM` | Чтение из памяти | 10 | 4 байта | `адрес` |
| `WRITE_MEM` | Запись в память | 7 | 2 байта | `адрес` |
| `BITREVERSE` | Битовое обращение | 1 | 2 байта | - |

## Примеры
```asm
LOAD_CONST 255     ; Загрузить константу 255
READ_MEM 1024      ; Прочитать память по адресу 1024
WRITE_MEM 2048     ; Записать в память по адресу 2048
BITREVERSE         ; Обратить биты аккумулятора
```
## Запуск первого этапа
`python etap1.py program1.asm output.txt --test`
## Запуск второго этапа
`python etap2.py program1.asm program2.bin --test`
## Запуск третьего этапа
`python etap3.py program3.bin memory_dump3.json --start 0 --end 4`
## Запуск четвертого этапа
`python etap2.py program4.asm program4.bin --test`\
`python etap4.py program4.bin memory_dump4.json --start 0 --end 4`
